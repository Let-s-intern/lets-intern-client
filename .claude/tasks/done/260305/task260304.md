## 아키텍처 원칙: 멘토 도메인 독립 분리

멘토는 user, admin과 **동급의 핵심 유저 타입**으로 독립 분리한다.

```
app/
├── (user)/     # 일반 유저 (취준생)     → Providers + ConditionalLayout
├── admin/      # 관리자                 → AdminProviders + AdminGuard + AdminSidebar
└── mentor/     # 멘토 ★ 독립 라우트     → MentorProviders + MentorGuard + MentorSidebar
```

admin 구조를 그대로 따라가는 패턴:
- `app/admin/layout.tsx` → `app/mentor/layout.tsx`
- `app/admin/AdminGuard.tsx` → `app/mentor/MentorGuard.tsx`
- `app/admin/AdminSidebar.tsx` → `app/mentor/MentorSidebar.tsx`
- `context/AdminProviders.tsx` → `context/MentorProviders.tsx`
- `domain/admin/` → `domain/mentor/`
