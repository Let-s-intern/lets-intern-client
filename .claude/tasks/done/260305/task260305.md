| **상태** | 🔲 pending |

| **차단** | #2 |

**현재 상태**: `FeedbackEditor.tsx`가 단순 `<textarea>`로 구현됨
**목표**: 어드민에서 사용 중인 `EditorApp` (Lexical) 에디터를 멘토 피드백에도 적용

**재사용 컴포넌트**:
- `EditorApp` — `src/domain/admin/lexical/EditorApp.tsx`
- `LexicalContent` — `src/domain/blog/ui/LexicalContent.tsx`
- `emptyEditorState` — EditorApp에서 export하는 빈 에디터 상태

**작업 내용**:

1. `FeedbackEditor.tsx` 수정
   - `<textarea>` → `EditorApp` 교체
   - Props 변경: `initialContent: string` → `initialEditorStateJsonString?: string`
   - `onChange` 콜백: Lexical JSON string 전달
   - `readOnly` 상태일 때: `LexicalContent`로 렌더링 (어드민 `ChallengeFeedbackPage.tsx:127-152` 패턴 참고)

2. `FeedbackModal.tsx` 수정
   - `editorContent` / `serverContent` 타입이 Lexical JSON string으로 동작하도록 조정
   - 초기값이 없을 때 `emptyEditorState` 사용

**참고 패턴** (어드민 구현):
```tsx
// ChallengeFeedbackPage.tsx:127-152
const FeedbackEditorApp = ({ initialEditorStateJsonString, onChange }) => {
  const { isCompleted } = useIsCompleted();
  if (isCompleted && initialEditorStateJsonString) {
    return <LexicalContent node={JSON.parse(initialEditorStateJsonString).root} />;
  }
  return <EditorApp initialEditorStateJsonString={initialEditorStateJsonString} onChange={onChange} />;
};
```

**수정 파일**:
- `src/domain/mentor/feedback/FeedbackEditor.tsx`
- `src/domain/mentor/feedback/FeedbackModal.tsx`
