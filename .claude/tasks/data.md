# 💬 현직자 피드백 상세 안내

혼자 하는 취업 준비는 끝, **렛츠커리어 현직자 멘토단**의 정밀 진단으로 완성도를 높이세요!

- 렛츠커리어 현직자 멘토단이란?
  **대기업 / IT 대기업 / 시리즈 B 이상의 스타트업 현직자**분들이 함께 렛츠커리어 멘토단으로 활동하고 있습니다!
  피드백 플랜 참여 시, 렛츠커리어 현직자 멘토단이 취준생 여러분께 알찬 피드백을 제공해드려요!
